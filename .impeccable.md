# Design Context

## Users
Professional users (judges, demo audience, productivity-focused individuals) who want a serene, high-end AI assistant interface. They're using it on laptop screens in demo or work settings, seeking a calm and capable tool that feels luxurious and trustworthy.

## Brand Personality
**Voice**: Confident but understated. Professional warmth without artificial cheer.
**3 words**: calm, elegant, precise
**Emotional goals**: Reassurance, clarity, sophistication. The user should feel at ease and in control.

## Aesthetic Direction
**Visual tone**: Calm luxury. Like a premium lounge or a fine timepiece — quiet, elegant, reassuring.
**Metaphor**: Fine parchment or linen — natural, tactile, timeless. Or a luxury watch face: minimalist markers, purposeful details.
**Theme**: Dark-only (deep blue-gray, not pure black).
**Color scheme**: Rich, muted, warm. Deep blue-gray backgrounds (oklch(0.12 0.01 255)), surfaces with subtle warmth, borders that define without shouting. Accent in a muted gold/bronze (oklch(0.75 0.07 75)). Zero gradients, zero neon, zero glow effects.
**Typography**: Aleo (serif) for headings — elegant, literary, confident. Manrope (sans-serif) for body — clean, modern, highly legible. JetBrains Mono for agent activity/logs.
**Motion**: Slow, smooth, easing out over ~250-300ms. Panel entries fade and slide gently. Stagger children for a composed entrance. No bounces, no elastic easing. Respect reduced-motion.

**Anti-references**: Any AI-generated "slop" — purple-to-blue gradients, gradient text, side-stripe borders, generic rounded cards with drop shadows, glassmorphism, sparklines, excessive glow. Avoid looking like Vercel/Linear/Perplexity clones.
**Differentiation**: Every pixel feels intentional. This is the anti-vibe-coded interface. Minimalist but not bland; refined but not cold.

## Design Principles
1. **No gradients anywhere** — solid colors only for text, backgrounds, accents.
2. **Intentional typography** — Aleo + Manrope pairing creates a sophisticated, literary quality; never use Inter, DM Sans, or other default sans.
3. **Quiet luxury** — muted colors, fine borders (1px), no shadows or glows. The design feels expensive through restraint, not ornamentation.
4. **Motion with purpose** — slow, eased, staggered entrances convey craftsmanship without being flashy.
5. **Content-first spacing** — varied rhythm using 4px scale; generous whitespace; max-width 65-75ch for readability.

## Technical Constraints
- Framework: Next.js + React, Tailwind CSS
- Target: Laptop/desktop demo (1024px+); mobile adaptation secondary
- Accessibility: Not a primary focus for this demo phase
- Logo: No mark yet; placeholder text "Clarity" in Aleo
